# Retail Sales Performance Analysis using SQL & Power BI

## 📊 Project Overview

This project analyses retail sales data using a SQL-based data warehouse and Power BI dashboards. The goal is to transform raw transactional data into meaningful business insights.

---

## 🎯 Objectives

* Design a star schema data warehouse
* Perform data cleaning and transformation using SQL
* Build an analytical model in Power BI
* Generate business insights from retail data

---

## 🛠️ Tools & Technologies

* Python (Pandas, Matplotlib)
* SQL (MySQL)
* Power BI
* Jupyter Notebook

---

## 📂 Dataset

Online Retail Dataset containing:

* Invoice details
* Product information
* Customer data
* Country and date

---

## 🔍 Key Insights

* Total Revenue: £10.67M
* Top 20% customers contribute ~60% revenue
* UK contributes ~85% of total sales
* Strong seasonal trends in Q4

---

## 🏗️ Data Architecture

Raw Data → Cleaning → SQL Data Warehouse → Star Schema → Power BI Dashboard
